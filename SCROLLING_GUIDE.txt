================================================================================
                          HOW TO SCROLL
================================================================================

DESKTOP (MOUSE):
  1. Mouse Wheel: Scroll up/down
  2. Scrollbar Drag: Click and drag scrollbar thumb
  3. Scrollbar Click: Click on track to jump
  4. Keyboard: Arrow keys, Page Up/Down, Home/End

MOBILE (TOUCH):
  1. Swipe: Swipe up/down to scroll
  2. Momentum: Swipe and release for momentum scrolling (iOS)
  3. Scrollbar: Tap and drag scrollbar (if visible)

TABLET (TOUCH + MOUSE):
  1. Touch: Swipe to scroll
  2. Mouse: Use mouse wheel or scrollbar
  3. Keyboard: Use arrow keys or Page Up/Down

================================================================================
                          KEYBOARD SHORTCUTS
================================================================================

Arrow Up:     Scroll up (small increment)
Arrow Down:   Scroll down (small increment)

Page Down:    Scroll down (page height)
Home:         Scroll to top

Tab:          Navigate to next field (also scrolls into view)
Shift+Tab:    Navigate to previous field (also scrolls into view)

================================================================================
                          TROUBLESHOOTING
================================================================================

ISSUE: Form not scrolling
FIX:
  1. Check if content exceeds viewport height
  2. Verify overflow-y: auto is set
  3. Check if flex: 1 is applied
  4. Verify modal-drawer has height: 100%

ISSUE: Scrollbar not visible
FIX:
  1. Check scrollbar CSS is applied
  2. Verify browser supports scrollbar styling
  3. Check scrollbar width is 8px
  4. Try hovering over scrollbar area

ISSUE: Scrolling is jerky
FIX:
  1. Verify scroll-behavior: smooth is set
  2. Check -webkit-overflow-scrolling: touch is set
  3. Verify no JavaScript scroll listeners
  4. Check browser performance

ISSUE: Scrollbar styling not working
FIX:
  1. Verify ::-webkit-scrollbar selectors
  2. Check scrollbar-width for Firefox
  3. Verify scrollbar-color for Firefox
  4. Check browser compatibility
