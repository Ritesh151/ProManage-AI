================================================================================
                          PROJECT ID FORMAT
================================================================================

Format: PF-YYYY-XXXX
Example: PF-2026-1001

Components:
  PF = ProposalForge prefix
  YYYY = Current year
  XXXX = Sequential number (starts at 1001)

Generation Logic:
  1. Get current year
  2. Find last project with same year prefix
  3. Increment number or start at 1001
  4. Format as PF-YYYY-XXXX
